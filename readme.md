# Waybar Khal

Simple waybar module that formats a khal calendar into a tooltip.

Default config:

- Events included in the nexy 14 days
- Calendar size set to 1 month in ~/.config/khal/config

```
# ~/.config/khal/config
[view]
min_calendar_display = 1
```
